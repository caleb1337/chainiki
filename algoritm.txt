Создать сайт по продаже чайников.
Сайт должен уметь работать с бд (MySQL). 
На странице должно быть реализовано отображение/редактирование/удаление чайника через ajax (можно использовать js). 
В визуальном плане - причесать, чтобы не голая таблица была, но и сильно стараться над этим не стоит
Обязательно backend должен быть на PHP
Не разрешается использование фраемворков, кроме jquery

Алгоритм

1. Главная страница, загружает данные из БД с чайниками
2.  Div с чайником состоит из след блоков:
    а) описание и артикль и цена чайника из БД
    б) статика, кнопки, которые будет обрабатывать JS и подготовленная разметка
3. Во вставленный код надо добавить кнопки для редактирования параметров через ajax
 

 БД структура столбцов

 1. ID автоинкремент
 2. Название чайника
 3. Артикль чайника уникальный
 4. Цена чайника
